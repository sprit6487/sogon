# 소곤 · SOGON

한국인과 외국인을 잇는 글로벌 매칭 앱 **소곤(SOGON)** 의 제품 시안(HTML 목업)입니다.
실제 제품은 이 시안을 기준으로 React Native로 구현될 예정이며, 본 저장소는 화면·인터랙션을
거의 정확하게 표상하기 위한 고충실도 프로토타입입니다. (백엔드 없음, 목 데이터로 동작)

## 타깃
- 한국인: 30대 후반 ~ 40대 후반
- 외국인: 30대 초반 ~ 40대 초반
- 키워드: 진지한 관계 · 안정 · 크로스컬처

## 브랜드
- 프라이머리 컬러: **클리어 로제 와인 `#DB6A85`**
- 로고: 속삭임 말풍선 (소곤 = 귓속말 컨셉)

## 파일
- [`index.html`](index.html) — 소곤 앱 프로토타입 (메인)
- [`logo-options.html`](logo-options.html) — 귓속말 로고 후보 8안
- [`colors.html`](colors.html) — 프라이머리 컬러 비교

## 주요 화면 / 기능
- **탐색** — 사진·프로필 카드. 스와이프 / 버튼 모두 지원
  - 직전 취소(💎 유료) · 넘기기 · 친구 신청 · 슈퍼라이크(💎 유료, 수락 없이 즉시 대화방)
  - 사진 하단을 끌어올리면 자기소개 시트
  - 카드 우측 상단 신고(사유 선택, 기타 직접 입력)
- **받은 신청** — 나를 선택한 사람들을 격자로, 수락/거절
- **대화** — 수락 시 대화방 생성. 메시지·답글·이미지·동영상·파일·음성, 대화 내 검색
- **프로필** — 사진·관심사·자기소개·기본 정보 편집

## 로컬 실행
정적 HTML이라 파일을 바로 열거나, 간단한 서버로 띄우면 됩니다.

```bash
python -m http.server 8000
# http://127.0.0.1:8000/index.html
```
